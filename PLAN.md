# 軌道衛星パンチ 現行仕様

この文書は、現在のソースコードを基準にしたゲーム仕様である。将来案は含めるが、実装済みの挙動と未実装の構想を分けて扱う。

## ゲーム概要

プレイヤーは惑星の周囲を自動周回する小型衛星を操作し、外向きにパンチを放って接近する脅威を弾き返す。弾き返した脅威を別の脅威へぶつけるとチェーンヒットが発生し、スコアと惑星シールド回復につながる。

現在の敗北条件は惑星HPが0になることのみ。プレイヤー本体に脅威が当たった場合は、対象を弾き返しつつ一時的なパンチロックアウトと無敵時間が発生する。

## 実装スタック

- Runtime: Phaser 3
- Language: TypeScript
- Build tool: Vite
- Canvas: 960 x 640、Phaser Scale FIT / CENTER_BOTH
- UI: DOMベースのHUDとオーバーレイ
- Main entry: `src/main.ts`
- Core simulation: `src/game/simulation.ts`
- Rendering: `src/game/GameScene.ts` と `src/game/rendering/*`
- Threat config: `src/game/data/threat-waves.json`

## 画面とUI

`index.html` はゲームシェル、HUD、オーバーレイを持つ。

- HUD表示:
  - `PLANET`: 惑星HPバー
  - `SCORE`: 現在スコア
  - `WAVE`: 現在ウェーブ
  - `READY`: パンチクールダウン状態。使用可能なら `OK`、待機中は残り割合を表示する。
- オーバーレイ:
  - タイトル画面
  - ポーズ画面
  - ゲームオーバー画面
  - Start / Resume / Retry ボタン
- チェーンヒット時はキャンバス上に `N hits!` と、回復があれば `+N shield` を表示する。

## 入力

現在の操作はワンボタン中心。

- `Space`: 外向きパンチ
- マウス左クリック / タップ: 外向きパンチ
- `Esc`: ポーズ
- オーバーレイボタン: 開始、再開、リトライ

現在は未実装:

- チャージショット
- `A` / `D` や左右キーによる周回速度操作
- プレイヤーの任意移動

## プレイヤーとパンチ

プレイヤーは惑星中心の周囲を自動周回する。

- 惑星中心: 画面中央
- 惑星半径: 62
- プレイヤー軌道半径: 136
- プレイヤー当たり判定半径: 16
- 周回速度: 1.95 rad/s

パンチはプレイヤー位置から外向きに発射される。

- クールダウン: 0.56秒
- 射程: 120
- パンチ半径: 18
- 伸長速度: 480
- 戻り速度: 560
- 最大到達後の保持時間: 0.24秒
- 判定: パンチ先端と、根元から手首付近までの線分判定

パンチが命中した脅威は外向きまたは接触方向へノックバックされる。ミニボスは通常脅威と違い、命中ごとにHPを削る。

## 脅威タイプ

現在の `ThreatKind` は5種類。

- `meteor`
  - 基本の隕石。惑星方向へ侵入し、パンチで弾き返せる。
- `orbitalSatellite`
  - 画面外から入り、惑星周辺を楕円軌道で横切る衛星。軌道フェーズが終わると消滅する。
- `explosiveCore`
  - 爆発コア。パンチ、チェーン衝突、惑星衝突で爆発し、周囲の脅威を巻き込む。
- `tractorDrone`
  - 牽引ドローン。範囲内の通常脅威の進行方向を惑星側へ曲げる。
- `miniBoss`
  - ミニボス。ウェーブ突入後にスケジュール出現し、渦巻き軌道で接近する。HPを持ち、複数回の命中が必要。

## 衝突とチェーン

パンチを受けた脅威は `knocked` 状態になり、別の脅威へ衝突するとチェーンが伸びる。

- 通常脅威へのチェーン衝突:
  - 対象を追加で弾き飛ばす。
  - チェーン数を増やす。
  - スコアボーナスを発生させる。
  - 惑星HPが最大未満ならシールドを回復する。
- 回復量:
  - 基本 `4 + chain`
  - 1回あたり最大12
  - 惑星HP上限は100
- 爆発コア:
  - 爆発半径は118。
  - 範囲内の脅威をノックバックまたは加速する。
  - チェーン表示と回復対象になる。
- ミニボス:
  - パンチまたは衛星本体衝突で1ダメージ。
  - チェーン衝突では2ダメージ。
  - 命中後は短いヒットクールダウンを持つ。

## プレイヤー本体への衝突

プレイヤー本体が脅威に接触した場合、現在はプレイヤーHPやシールドを消費しない。

- 通常脅威:
  - 脅威を接触方向へ弾き返す。
  - パンチクールダウンを最大1.35秒まで延長する。
  - プレイヤーに1.35秒の無敵時間を付与する。
  - ヒットストップと画面揺れを発生させる。
- ミニボス:
  - ミニボスに1ダメージを与える。
  - 同じくロックアウトと無敵時間が発生する。

旧案にあった「プレイヤーシールドが0の状態で大型障害物に衝突すると敗北」は未実装。

## 惑星への衝突とゲームオーバー

惑星HPの最大値は100。脅威が惑星へ衝突するとHPが減る。

- `meteor`: 18ダメージ
- `orbitalSatellite`: 16ダメージ
- `tractorDrone`: 20ダメージ
- `miniBoss`: 38ダメージ
- `explosiveCore`: 爆発処理に加えて32ダメージ

惑星HPが0以下になるとゲームオーバー。ゲームオーバー時はオーバーレイにスコアと到達ウェーブを表示し、Retryできる。

## ウェーブとスポーン

ウェーブは撃破数を基準に進行する。

- 8撃破ごとに次ウェーブへ進む。
- `1 + floor(defeated / 8)` が現在ウェーブ。
- スポーン設定は `src/game/data/threat-waves.json` で管理する。
- 設定済みウェーブを超えた場合は、JSON内のウェーブ定義をループして使う。
- ランダムスポーンにミニボスは直接混ぜず、ミニボス重みがあるウェーブでは突入10秒後に1体を予約出現させる。

現在のウェーブ定義:

- Wave 1
  - 隕石と軌道衛星が半々で出現。
  - 爆発コア、牽引ドローン、ミニボスは出ない。
- Wave 2
  - 隕石、爆発コア、軌道衛星が出現。
  - 速度、狙い範囲、スコア倍率が少し上がる。
- Wave 3
  - 隕石、少量の爆発コア、軌道衛星、牽引ドローンが出現。
  - 牽引ドローンが本格的に混ざる。
- Wave 4
  - スポーン間隔が大きく短縮される。
  - 軌道衛星が多く、爆発コア、牽引ドローン、ミニボスも出現対象になる。

## スコア

スコアは脅威タイプ、ウェーブ、JSON側の倍率、チェーンや撃破ボーナスで決まる。

- 通常撃破時は `scoreForThreat` を使う。
- ミニボス撃破時は `defeatBonus` が加算される。
- チェーン衝突や爆発巻き込みでは追加スコアが入る。
- 現在、ハイスコア保存やリザルト詳細保存は未実装。

## ビジュアルとフィードバック

現在のゲーム内描画は Phaser Graphics によるプリミティブ描画が中心。

- 背景:
  - ゲームシェル背景に `assets/orbit-punch-key-art.png` を使用。
  - キャンバス内では暗い宇宙背景と星を描画。
- 惑星:
  - 海、陸、シールドリング、低HP時の亀裂を描画。
- プレイヤー:
  - 黄色の小型衛星。
  - 向き線と無敵中の点滅リングを表示。
- パンチ:
  - 伸びる腕、グローブ、保持中リングを描画。
- 脅威:
  - 隕石、軌道衛星、爆発コア、牽引ドローン、ミニボスを個別シルエットと色で描き分ける。
- フィードバック:
  - ヒットストップ
  - 画面揺れ
  - スパーク
  - チェーンヒットラベル
  - 牽引範囲と牽引方向ガイド

現在は未実装:

- 効果音、BGM、音量設定
- 外部スプライトアニメーション
- 粒子専用システム
